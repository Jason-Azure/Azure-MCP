# Azure-MCP

A collection of MCP (Model Context Protocol) servers for Azure services that work with GitHub Copilot.

## 🚀 Quick Start

### Installation

**中文用户**: 请查看 [中文安装指南](./INSTALL_zh-CN.md) 获取详细的安装说明。

**English users**: See [Installation Guide](./INSTALL.md) for detailed installation instructions.

### What is MCP?

MCP (Model Context Protocol) enables AI assistants like GitHub Copilot to securely connect with your development tools and services. This repository provides MCP servers specifically designed for Azure services.

## 📋 Prerequisites

- Node.js 18 or higher
- GitHub Copilot subscription
- Azure account with appropriate permissions

## 📦 Available Servers

This repository aims to collect and provide MCP servers for various Azure services:

- **Azure Resource Manager** - Manage and query Azure resources
- **Azure Storage** - Interact with Azure Blob Storage, Files, and more
- **Azure AI Services** - Access Azure OpenAI and Cognitive Services
- More coming soon...

## 🔧 Quick Installation

```bash
# Install via npm
npm install -g @modelcontextprotocol/server-azure

# Or run directly with npx
npx @modelcontextprotocol/server-azure
```
